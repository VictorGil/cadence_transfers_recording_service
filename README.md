# Cadence Transfers Recording Service
  
This is an example/PoC of how to use **Uber Cadence** in order to implement event-driven microservices,
together with Apache Kafka.  
  
## Requirements
 - Maven 3  
 - Java 12  
 - Uber Cadence
 - Confluent Platform (which includes Kafka)
